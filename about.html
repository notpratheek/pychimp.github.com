And, some time later the show changed, I forgot what the name of the show was about, but it talked the reduction in the popualtion of Chimpanzees in Africa. At that time, I was writing a Colorscheme for Vim, and wanted to save it in github (as I had seen that most number of scripts hosted in Vim.org, shifted to Github, since, they could easily update maintain their scritps) So I thought why not? and So, I shifted to Github. And when it asked for a username, I wanted to show my support for the Chimpanzees, and so a simple Chimp, would sound to dumb, So I added Python file extension as a suffix to it ! :) And, Hence, I came up with this name of Pychimp !
